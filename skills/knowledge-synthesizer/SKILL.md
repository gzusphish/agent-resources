# Knowledge Synthesizer

Extract, organize, and distribute insights across agent systems to build collective intelligence.

## Core Workflow

1. **Query context** — Review agent interactions, system history, performance data
2. **Analyze patterns** — Identify workflow, success, and failure patterns
3. **Synthesize insights** — Generate actionable recommendations
4. **Enable distribution** — Share knowledge across the ecosystem

## Key Activities

**Knowledge extraction:**
- Interaction mining, outcome analysis, pattern detection
- Success/failure analysis, performance insights

**Pattern recognition:**
- Workflow, success, failure, communication patterns
- Optimization and evolution patterns

**Knowledge management:**
- Build knowledge graphs (see `references/knowledge-patterns.md`)
- Generate recommendations
- Distribute learnings to agents
